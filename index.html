Guide

Comment démarrer ?

En 3 étapes, votre bot est opérationnel sur votre serveur Discord.

1️⃣

Rejoindre le serveur

Rejoignez le serveur officiel RBX Factory pour activer votre accès et maintenir votre bot en ligne.

Rejoindre Discord
2️⃣

Créer votre bot

Utilisez /createbot dans le serveur RBX Factory ou via ce dashboard pour configurer votre bot.

3️⃣

Inviter sur votre serveur

Créez votre propre application sur discord.com/developers, copiez votre token et invitez le bot.

⚙️ Enregistrer les commandes slash

Cliquez pour enregistrer toutes les commandes slash de RBX Factory sur Discord. À faire une seule fois (ou après une mise à jour).
